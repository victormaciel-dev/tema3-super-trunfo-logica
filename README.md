# Tema 3 — Super Trunfo com Lógica (C)

Projeto desenvolvido em C que simula comparações entre cartas do jogo Super Trunfo utilizando estruturas de decisão.

## Conceitos aplicados
- if / else
- decisões encadeadas
- switch
- operador ternário
- menus interativos
- comparação de atributos

## Funcionalidades
- cadastro de duas cartas
- exibição das cartas
- comparação por atributo escolhido
- comparação geral considerando múltiplos atributos

## Compilar e executar
```bash
gcc super_trunfo.c -o jogo
./jogo
